FILES TO UPDATE:
──────────────────────────────────────────────────────────────
After adding a new business, you MUST update these files:

☐ 1. js/data.js - Add business entry
  ☐ Use coordinates: { lat: X, lng: Y }
  ☐ If in mall: Add locatedInMall: "mall-id"
  ☐ If in mall: Update mall's tenants array with your business ID

☐ 2. js/config.js - Add search keywords
  ☐ Add business name and aliases to searchAliases
  ☐ Bump version number (e.g., 4.3.2 → 4.3.3)

☐ 3. index.html - Update business counts (3 places)
  ☐ Hero stats section: data-target="30" → data-target="31"
  ☐ Map overlay section: "30 listings" → "31 listings"
  ☐ Stats grid section: data-target="30" → data-target="31"

☐ 4. categories.html - Update category count
  ☐ Find your business category
  ☐ Increment the listing count (e.g., "4 listings" → "5 listings")

☐ 5. Update version numbers in these files:
  ☐ js/main.js - @version comment
  ☐ js/pwa.js - @version and PWA_VERSION constant
  ☐ manifest.json - version field
  ☐ sw.js - @version, CACHE_VERSION, BUILD_NUMBER

☐ 6. sw.js - Update STATIC_ASSETS list if new files added
