# Property Assets

This folder contains property images and videos.

## Structure

```
property/assets/
├── property-1/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── video.mp4
├── property-2/
│   └── ...
└── placeholder.svg

property/voices/
├── property-1/
│   └── description.mp3
├── property-2/
│   └── description.mp3
└── ...
```

## Adding Property Media

### Images and Videos
1. Create a folder with the property ID in `property/assets/` (e.g., `property-1`)
2. Add property images (JPG/PNG)
3. Add property videos (MP4) - optional

### Voice Descriptions
1. Create a folder with the property ID in `property/voices/` (e.g., `property-1`)
2. Add voice description audio named `description.mp3`

## Voice Descriptions

Voice descriptions should be named `description.mp3` and placed in the property voices folder.

Example: `property/voices/property-1/description.mp3`
