# Property Voice Descriptions

This folder contains voice descriptions for properties.

## Structure

```
property/voices/
├── property-1/
│   └── description.mp3
├── property-2/
│   └── description.mp3
└── ...
```

## Adding Voice Descriptions

1. Create a folder with the property ID (e.g., `property-1`)
2. Add the voice description audio file named `description.mp3`

## Audio Format

- Format: MP3
- Recommended bitrate: 128kbps or higher
- Mono or Stereo
- File name: `description.mp3` (case-sensitive)

## Example

For a property with ID `property-1`:
- Path: `property/voices/property-1/description.mp3`

The audio will be automatically loaded when users click the listen button on the property detail page.

## Recording Tips

- Speak clearly and at a moderate pace
- Include key property details: location, size, price, amenities
- Keep descriptions between 1-3 minutes
- Use a quiet environment for recording
- Consider professional voice-over for better quality
